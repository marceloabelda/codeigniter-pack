Configuración
-----
* autoload:

	$autoload['libraries']	= array('database', 'session', 'template', 'firephp');

* config: 

	$config['language']	= 'spanish'; 

	paso BASE_URL como constante

* template: 

	$config['parser_body_enabled'] = TRUE;
